# План запуска и автоматизации SEO-сайта

## Общая архитектура решения

**Домен и хостинг**
- Домен: купить у регистратора (или бесплатно на Freenom).
- Хостинг: Vercel/GitHub Pages (static), либо бесплатный хостинг WordPress.com (используя плагин развертывания скриптом).

**Стек технологий**
- Язык: Python 3.
- Генерация статей: OpenAI ChatGPT API (или локальная LLM через Hugging Face).
- Сбор ключевых слов: Google Keyword Planner API / SerpAPI (по бесплатному тарифу).
- Шаблонизатор: Jinja2 (для статических страниц) или WP-CLI (для WordPress).
- Оркестрация: cron + GitHub Actions/Vercel Deploy Hooks.
- Аналитика: Google Analytics 4 + Google Search Console.
- Монетизация: Google AdSense и партнёрки (Amazon Associates, AliExpress, Travelpayouts и пр.).

## Поэтапный план на 7 дней
| День | Задачи | Результат |
|------|--------|-----------|
| 1 | Настройка инфраструктуры | Домен, репозиторий, базовый шаблон |
| 2 | Подключение API и сбор ключевых слов | Скрипт сбора 50 нишевых фраз |
| 3 | Генерация первых статей | 5–10 черновых статей в Markdown/HTML |
| 4 | Верстка и публикация MVP | Авто-деплой первой версии сайта |
| 5 | Монетизация | Подключены скрипты AdSense и партнёрок |
| 6 | Настройка автогенерации | cron/GitHub Actions запускают генерацию |
| 7 | Тест и мониторинг | Проверка логов, Analytics, Search Console |

## Детализация по дням

### День 1: Инфраструктура и шаблон
- Регистрация домена и создание репозитория на GitHub.
- Инициализация проекта, установка зависимостей (openai, jinja2, requests, PyYAML).
- Создание базового шаблона (Jinja2): `templates/article.html`.
- Файл конфигурации `config.yml` (ключи API, параметры SEO, партнёрские ID).

### День 2: Сбор и фильтрация ключевых слов
- Подключение к SerpAPI или Google Keyword Planner.
- Скрипт для получения и фильтрации ключей, сохранение в `keywords.csv`.

### День 3: Генерация контента через ChatGPT
- Генерация статей по ключевым словам, сохранение в Markdown (`content/{slug}.md`).

### День 4: Верстка и деплой MVP
- Конвертация Markdown → HTML с помощью Jinja2, сохранение в `public/`.
- Деплой на GitHub Pages или Vercel.

### День 5: Подключение монетизации
- Вставка AdSense-кода и партнёрских ссылок в шаблон.

### День 6: Автоматизация генерации и публикации
- Настройка cron или GitHub Actions для ежедневного запуска генерации и деплоя.

### День 7: Тестирование и мониторинг
- Проверка логов, подключение Google Analytics и Search Console.

## Приложения

### Пример config.yml
```yaml
openai_key: "sk-..."
serpapi_key: "..."
amazon_id: "your-tag-20"
adsense_code: "<script>...</script>"
article_per_run: 5
```

### Структура проекта
```
auto-seo-site/
├── config.yml
├── content/           # сгенерированные .md
├── templates/
│   └── article.html
├── public/            # сгенерированные .html
├── generate_and_deploy.py
├── requirements.txt
└── logs/
```

### Примеры команд
- `python generate_and_deploy.py` — полный цикл «сбор → генерация → деплой».
- `git push origin gh-pages` — обновление GitHub Pages.
