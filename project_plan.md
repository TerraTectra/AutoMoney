# Актуальный план развития SEO-сайта

## Архитектура и принципы
- Полностью бесплатная реализация: только open-source, без платных API.
- Хостинг: GitHub Pages (статический сайт).
- Генерация: Python 3 + Jinja2, локальная LLM (Ollama) или заглушки.
- Ключевые слова: вручную или через бесплатные источники, keywords.csv.
- Монетизация: партнёрские ссылки, AdSense (по желанию).
- Автоматизация: GitHub Actions.

## Приоритеты и этапы

1. **Генерация реальных статей**
   - Подключить Ollama или другую локальную LLM для генерации уникального текста.
   - Заменить заглушки на полноценные статьи.

2. **SEO-улучшения**
   - Добавить sitemap.xml и robots.txt для лучшей индексации.
   - Генерировать метатеги (title, description) для каждой статьи.

3. **Монетизация**
   - Вставить реальные партнёрские ссылки (Amazon, AliExpress и др.).
   - Подключить AdSense (если планируется).

4. **Автоматизация**
   - Настроить GitHub Actions для ежедневной генерации и деплоя новых статей.
   - (Опционально) Автоматическое обновление keywords.csv.

5. **Улучшение дизайна**
   - Доработать шаблон index.html и article.html: добавить превью, стили, мобильную адаптацию.
   - Добавить favicon, логотип, кастомный footer.

6. **Аналитика**
   - Подключить Google Analytics и Search Console для отслеживания трафика.
   - Добавить логи и мониторинг ошибок.

## Детализация ближайших шагов

### 1. Генерация реальных статей
- Настроить Ollama или выбрать другую бесплатную LLM.
- Интегрировать генерацию реального текста в generate_and_deploy.py.
- Проверить качество и уникальность статей.

### 2. SEO-улучшения
- Сгенерировать sitemap.xml и robots.txt автоматически.
- Добавить генерацию метатегов для статей.

### 3. Монетизация
- Вставить реальные партнёрские ссылки в шаблон.
- Подключить AdSense, если есть одобрение.

### 4. Автоматизация
- Проверить и доработать GitHub Actions для ежедневного автодеплоя.
- (Опционально) Автоматическое обновление keywords.csv.

### 5. Улучшение дизайна
- Доработать шаблоны, добавить стили, favicon, логотип.

### 6. Аналитика
- Подключить Google Analytics и Search Console.
- Добавить логи ошибок и мониторинг.

### Примеры команд
- `python generate_and_deploy.py` — полный цикл «сбор → генерация → деплой».
- `git push origin gh-pages` — обновление GitHub Pages.
